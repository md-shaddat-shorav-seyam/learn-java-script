In **JavaScript**, there are two main categories:

1. **Built-in (Core) Objects** → already available in JS
2. **External Libraries** → you install them (like npm packages)

Since you asked like **Math, String, Date**, those are **built-in objects**, not libraries.

---

# ✅ 1️⃣ JavaScript Built-in Objects (Core)

JavaScript does not call them "libraries" — they are **global objects**.

Here are the most important ones:

## 🔢 Numbers & Math

* `Number`
* `BigInt`
* `Math`

## 🔤 Text

* `String`
* `RegExp`

## 📅 Date & Time

* `Date`

## 📦 Collections

* `Array`
* `Map`
* `Set`
* `WeakMap`
* `WeakSet`

## 🧠 Structured Data

* `Object`
* `JSON`

## ⚙️ Control & Error

* `Promise`
* `Error`
* `Symbol`
* `Proxy`
* `Reflect`

## 🌐 Browser APIs (Not pure JS, but available in browsers)

* `fetch`
* `localStorage`
* `document`
* `window`
* `console`

---

# 📊 Total Built-in Core Objects?

There are **40+ standard built-in objects** in modern JavaScript (ES2024).

But the main important ones are around **20–25** that developers use daily.
